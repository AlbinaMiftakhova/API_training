API_training

--------------------------------------

API для работы со списком доступных в магазине книг. Позволяет:

- получить список доступных книг;
- получить сущность книги по ее идентификатору;
- добавить книгу в список доступных;
- удалить книгу по ее идентификатору;
- отсортировать список книг по названию.

---------------
Спрятан пароль от базы данных (использовался Security manager tool)
